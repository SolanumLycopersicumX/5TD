# 大资产与 Git LFS

Large project assets are tracked through Git LFS rather than copied into the knowledge vault.

## Asset Types

- Archives.
- Model weights.
- Videos.
- Databases.
- Raw or derived large data files.

## Vault Rule

The vault should index and link asset locations only. It should not embed binary assets or duplicate image/video/model files.
