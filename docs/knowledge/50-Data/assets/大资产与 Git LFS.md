# 大资产与 Git LFS

大型项目资产通过 Git LFS 管理，不复制到知识库中。

## 资产类型

- 压缩包。
- 模型权重。
- 视频。
- 数据库。
- 原始或派生的大型数据文件。

## Vault 规则

知识库只索引和链接资产位置，不嵌入二进制资产，也不复制图片、视频或模型文件。
