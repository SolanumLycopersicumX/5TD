# 当前优先级与路线决策

当前优先级是保持 RGB-only HBD-Net-RT 基线可运行，在真实隧道条件下验证它的安全性；只有当 RGB-only 无法提供足够安全证据时，才补充测距或融合路线。

## 当前顺序

1. 保持 [[HBD-Net-RT Baseline|HBD-Net-RT 基线]] 可运行。
2. 评估 RGB-only 对可通行区域、障碍物、工人、工程车辆和 [[Hard Boundary]] 的检测能力。
3. 如果 RGB 无法安全验证沟边距离，则加入右侧测距、LiDAR 或融合方案。
4. RL、diffusion、VLM 和 Transformer 工作必须位于 [[Costmap 与 Risk Grid]]、[[DWAPlanner]] 和 [[Safety Filter]] 之后。

## 决策含义

这个仓库既不是 fusion-only，也不是 legacy-only。可交付的近期 MVP 是 RGB-only；增强研究路线用于安全升级或性能提升。
