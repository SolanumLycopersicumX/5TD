# 入口脚本索引

The active baseline exposes script-level entrypoints for full pipeline, dashboard, inference, planning demo, latency benchmark, and visualization.

## Scripts

- `scripts/run_pipeline.py`: full frame-to-command pipeline.
- `scripts/run_dashboard.py`: multi-panel debug view.
- `scripts/run_inference.py`: perception-only inference.
- `scripts/run_planner_demo.py`: DWA-only planning demo.
- `scripts/benchmark_latency.py`: per-stage timing benchmark.
- `scripts/visualize_outputs.py`: static report image generation.
