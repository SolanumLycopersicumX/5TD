# 测试矩阵

基线测试套件覆盖配置、模型输出形状、后处理格式、栅格生成、DWA 行为、安全状态转换和场景决策。

## 测试范围

- 配置加载。
- 模型输出维度。
- 后处理输出格式。
- BEV、占用栅格和风险栅格规则。
- DWA 场景：空旷直行、障碍绕行、硬边界拒绝、无路停车。
- 安全状态转换和强制零速。
- 端到端决策场景。

## 命令

```bash
cd baselines/hbdnet_rt
export PYTHONPATH=src
pytest tests/ -v
```

## 环境注意

完整 torch 相关测试需要安装 `torch`。项目进展日志记录过，本地环境中非 torch 子集曾通过。
