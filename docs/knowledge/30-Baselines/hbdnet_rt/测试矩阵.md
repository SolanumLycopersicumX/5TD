# 测试矩阵

The baseline test suite checks configuration, model output shapes, postprocess format, grid generation, DWA behavior, safety-state transitions, and scenario decisions.

## Test Areas

- Config loading.
- Model output dimensions.
- Postprocess output format.
- BEV, occupancy, and risk-grid rules.
- DWA scenarios: open path, obstacle avoidance, hard-boundary rejection, no-path stop.
- Safety-state transitions and forced zero speed.
- End-to-end decision scenarios.

## Command

```bash
cd baselines/hbdnet_rt
export PYTHONPATH=src
pytest tests/ -v
```

## Environment Caveat

Full torch-dependent tests require `torch`. Previous project progress notes recorded a non-torch subset passing in the local environment.
