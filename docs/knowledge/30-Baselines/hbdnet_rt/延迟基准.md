# 延迟基准

The baseline latency target is end-to-end image-to-command latency below 100 ms.

## Current Baseline Record

The baseline docs report simulated current total latency around 36 ms mean and 37 ms P95, excluding real image acquisition and communication latency.

## Bottleneck

Model inference is the current main bottleneck in the placeholder network. Planned optimizations include a lighter backbone, ONNX/TensorRT FP16, vectorized DWA, and C++ migration for hot postprocess/BEV/DWA paths.
