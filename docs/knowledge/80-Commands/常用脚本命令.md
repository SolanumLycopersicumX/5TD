# 常用脚本命令

Common commands for the active HBD-Net-RT baseline.

## Full Pipeline

```bash
cd baselines/hbdnet_rt
export PYTHONPATH=src
python scripts/run_pipeline.py -n 50
```

## Dashboard

```bash
python scripts/run_dashboard.py
python scripts/run_dashboard.py --camera
```

## Single Modules

```bash
python scripts/run_inference.py
python scripts/run_planner_demo.py
python scripts/benchmark_latency.py -n 200
```

## Tests

```bash
pytest tests/ -v
python tests/test_scenarios.py
```
