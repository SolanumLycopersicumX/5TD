# 端到端运行流程

The runnable HBD-Net-RT pipeline turns one RGB frame into a safety-filtered control command.

```text
RGB frame -> [[Perception 模块]] -> [[Mapping 模块]] -> [[DWAPlanner]] -> [[SafetyStateMachine]] -> [[ControlCommand]]
```

## Stage Responsibilities

- [[Perception 模块]] converts the input image into detections, masks, and confidence values.
- [[Mapping 模块]] projects perception outputs into occupancy and risk grids.
- [[DWAPlanner]] samples candidate trajectories and scores feasible motion.
- [[SafetyStateMachine]] limits or stops motion based on confidence, risk, distances, and path feasibility.
- [[ControlCommand]] emits the final speed, steering, brake flag, safety state, and reason.
