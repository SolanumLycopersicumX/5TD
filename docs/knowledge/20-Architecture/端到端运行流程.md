# 端到端运行流程

可运行的 HBD-Net-RT 管线把一帧 RGB 图像转换成经过安全修正的控制命令。

```text
RGB frame -> [[Perception 模块]] -> [[Mapping 模块]] -> [[DWAPlanner]] -> [[SafetyStateMachine]] -> [[ControlCommand]]
```

## 阶段职责

- [[Perception 模块]]：把输入图像转换为检测、mask 和置信度。
- [[Mapping 模块]]：把感知输出投影成占用栅格和风险栅格。
- [[DWAPlanner]]：采样候选轨迹并评分可行动作。
- [[SafetyStateMachine]]：根据置信度、风险、距离和路径可行性限速或停车。
- [[ControlCommand]]：输出最终速度、转角、刹车标志、安全状态和原因。
