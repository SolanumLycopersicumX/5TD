# 安全约束导航

安全约束导航意味着感知和规划不能单独决定最终动作。最终命令必须经过风险、边界、置信度和路径可行性检查。

## 分层防护

1. [[Perception 模块]] 估计障碍物、可通行区域、硬边界、边缘和风险。
2. [[Mapping 模块]] 把硬边界和不可通行区域转成占用和风险。
3. [[DWAPlanner]] 拒绝与高风险区域碰撞的轨迹。
4. [[SafetyStateMachine]] 在正常、谨慎、减速、停车和人工接管之间升级。
5. [[ControlCommand]] 在 STOP 或 TAKEOVER 状态下强制零速和刹车。
