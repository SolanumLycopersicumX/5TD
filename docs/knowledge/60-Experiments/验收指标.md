# 验收指标

验收应同时覆盖导航、安全、模型和延迟指标。

## 指标范围

- 代表性隧道场景下的导航行为。
- 围绕 [[Hard Boundary]]、[[Trench Keep-out]]、工人和无路场景的安全行为。
- 感知输出的模型质量。
- 相对于 [[延迟基准]] 的延迟表现。

## 当前训练目标

HBD-Net-RT 训练指南记录的目标感知指标包括：detection mAP@0.5 不低于 75%，worker recall 不低于 90%，passable mIoU 不低于 88%，ditch boundary IoU 不低于 70%。
