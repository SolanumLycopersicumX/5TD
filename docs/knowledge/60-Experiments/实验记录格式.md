# 实验记录格式

实验报告应能横向比较 RGB-only、fusion、RL、diffusion 和 VLM 相关路线。

## 建议字段

- 日期和 commit。
- 路线：RGB-only、RGB temporal、LiDAR-only、LiDAR-RGB fusion、RL、diffusion 或 VLM。
- 数据集或场景。
- 使用的安全检查。
- 指标。
- 失败案例。
