# 部署边界

Deployment material belongs under `deployment/` after the runtime target is confirmed.

## Candidate Deployment Assets

- systemd services.
- Docker or container files.
- Edge-device setup notes.
- Watchdog and safety-startup scripts.

## Current Boundary

The current active baseline is runnable as scripts, but ROS 2 integration, real chassis control, and production deployment are not yet established. Final deployment must preserve [[Safety Filter]] behavior and command constraints from [[ControlCommand]].
