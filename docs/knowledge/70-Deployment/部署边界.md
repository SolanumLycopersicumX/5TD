# 部署边界

运行目标确认后，部署材料才进入 `deployment/`。

## 候选部署资产

- systemd 服务。
- Docker 或容器文件。
- 边缘设备配置说明。
- watchdog 和安全启动脚本。

## 当前边界

当前活跃基线以脚本方式可运行，但 ROS 2 集成、真实底盘控制和生产部署尚未确定。最终部署必须保留 [[Safety Filter]] 行为和 [[ControlCommand]] 的命令约束。
