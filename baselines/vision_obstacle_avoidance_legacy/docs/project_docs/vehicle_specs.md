## 运动学参数推算

### 编码器分辨率
```
编码器线数: 1024 P/R
4倍频: 4096 pulse/rev
减速比: 20:1
每轮转一圈: 4096 × 20 = 81920 pulses
若轮径 = 0.30m，周长 = 0.942m
每米脉冲数: 81920 / 0.942 ≈ 86964 pulses/m
每脉冲距离: 0.0115 mm/pulse （极高精度）
```

### 最大速度
```
16 km/h = 4.44 m/s
若轮径 = 0.30m，周长 = 0.942m
轮转速 = 4.44 / 0.942 = 4.71 rps = 283 rpm
电机转速 = 283 × 20 = 5660 rpm （合理范围）
```

### 最大角速度 (差速模式)
```
轮距 W ≈ 1.05m（待实测）
最大轮速差: v_R - v_L = 2 × 4.44 = 8.88 m/s (理论极限)
ω_max = 8.88 / 1.05 = 8.46 rad/s (理论)
实际安全限制: ~4.0 rad/s
原地旋转: ω = 2 × v / W, v=1.0m/s 时 ω≈1.9 rad/s
```

## 驱动方式分析

4×2000W 独立电机 + 直流无刷 → **四轮独立驱动**。
通过将左右两侧电机配对控制，可实现**差速驱动**（skid-steer）：
- 左侧 2 个电机 = v_L
- 右侧 2 个电机 = v_R
- 线速度 v = (v_L + v_R) / 2
- 角速度 ω = (v_R - v_L) / W
